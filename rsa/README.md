# Setup

Create virtual enviroment

~~~~ bash
python -m venv venv
~~~~

Activate virtual enviroment

~~~~ bash
./venv/Scripts/activate
~~~~

Install pip-tools

~~~~ bash
python -m pip install pip-tools
~~~~

Install denpencies

~~~~ bash
pip-sync
~~~~

# Usage

For usage help run

~~~~ bash
python -m simplersa --help
~~~~
