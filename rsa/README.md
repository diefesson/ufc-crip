# Setup

Create virtual environment

~~~~ bash
python -m venv venv
~~~~

Activate virtual environment

~~~~ bash
./venv/Scripts/activate
~~~~

Install package in editable mode with dev dependencies

~~~~ bash
python -m pip install -e .[dev]
~~~~

# Build

~~~~ bash
python -m build
~~~~

# Usage

For usage help run

~~~~ bash
python -m drsa --help
~~~~
